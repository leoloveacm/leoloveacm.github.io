AndroidLearning(1)——UI设计

chapter.1 控制UI界面

使用的是android 5.1.1版本，因为手机暂时是这个版本的。

Android提供了四种方法控制UI，分别是：

  1. XML布局文件
  2. 在代码中控制
  3. 使用XML和Java混合
  4. 开发自定义View
    下面我们开始了解这四种布局的方式

    1.1 使用XML布局控制UI

    在此之前需要提前了解XML的写法和Android解析XML的方式(为什么不用JSON，好写好看，orz)
    关键是下面两步：
  • 编写res/layout下的XML文件，注意命名规则需要符合Java的命名
  • 显示XML代码中布局的内容

在activity_main.xml中，通过设置不同的属性可以对于背景、组件样式、位置等等进行设置。
对于我们固定的一些文字内容，我们往往会加到string.xml中间
对于一些对于格式的调整，我们加到style.xml中(可能需要自己创建)

1.2 使用代码控制UI界面

android 支持想Swing那样完全通过代码控制UI界面，也就说可以用new来创建所有的组建。
关键是下面几步：

  • 创建布局管理器
  • 创建具体的组件
  • 将这个组件加到布局管理器中间
